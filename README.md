# CrazyDaily
一款程序员日常放松的App，基于Material Design + MVP-Clean + RxJava2 + Retrofit + Dagger2 + Glide + Okhttp + MTRVA + BRVAH + 其它炫酷的控件

## 效果图

* 干货集中营
* 天气
* 内涵段子
