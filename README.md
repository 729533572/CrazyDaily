# CrazyDaily
一款程序员日常放松的App，基于Material Design + MVP-Clean + Weex + RxJava2 + Retrofit + Dagger2 + Glide + Okhttp + MTRVA + BRVAH + 炫酷控件 + 炫酷动画

## 简介
* 知乎日报
* 干货集中营
* 天气
* 搞笑视频
* 联系人

本项目涵盖了开发中应该用到的开源框架、自定义控件、自定义动画、项目架构搭建。
